# HIV Immunology: CD4/CD8 T Cell Analysis

## Overview
This project analyses CD4+ and CD8+ T cell counts in HIV-positive patients 
compared to healthy controls using Python. The analysis demonstrates the 
characteristic immune dysfunction associated with HIV infection through 
statistical testing and data visualisation.

## Background
HIV progressively destroys CD4+ T cells, which are critical for coordinating 
the immune response. As CD4+ cells decline, CD8+ cytotoxic T cells increase 
as the immune system tries to fight the infection. The CD4/CD8 ratio is a 
key clinical marker for monitoring HIV progression.

## Key Findings
- HIV+ patients had significantly lower CD4 counts (322 cells/uL vs 882 cells/uL)
- CD8 counts were significantly higher in HIV+ patients (903 vs 498 cells/uL)
- CD4/CD8 ratio was dramatically reduced in HIV+ patients (0.37 vs 1.82)
- Both differences were statistically significant (p < 0.0001)

## Data
Simulated dataset based on published clinical reference ranges for HIV 
immunology research (n=60 patients: 30 HIV+, 30 healthy controls).
